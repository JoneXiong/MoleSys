PHP 设置项 说明
short_open_tag 是否要支持 PHP 代码的短开始标签 <?
expose_php 是否在 HTTP 头中包含 PHP 版本信息
max_execution_time 脚本最长执行时间 秒
memory_limit 单个PHP进程的内存限制 MB
display_errors 是否在出错时显示错误信息
post_max_size POST 数据大小限制 MB
upload_max_filesize 上传文件大小限制 MB
date.timezone 默认时区设置，格式如 Asia/Shanghai
FastCGI 设置项 说明
listen FastCGI 监听地址
pm 是否启用动态进程控制模型
pm.max_children 最大的 CGI 进程数量 个
pm.start_servers 起始的 CGI 进程数量 个
pm.min_spare_servers 最小允许的空闲 CGI 进程数量 个
pm.max_spare_servers 最多允许的空闲 CGI 进程数量 个
pm.max_requests 每个 CGI 进程执行多少次后重启进程 个
request_terminate_timeout 请求处理超时时间，超时后请求将被终止 秒
request_slowlog_timeout 请求处理超过多少秒后记入慢请求日志 秒
